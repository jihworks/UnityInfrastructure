# Cinemachine namespace

Some types in this namespace depends on Unity official `Cinemachine` package.

## Install Package

Install `Cinemachine` package from Unity Package Manager<i>(UPM)</i>.

> [!IMPORTANT]
> The Unity had made breaking changes to the Cinemachine version 3 from version 2.  
> This namspace dose **NOT** support version 2.

## Conditional Compile

Add `INFRASTRUCTURE_USE_CINEMACHINE` as symbol.
